# Tiny-SQLite-Database-using-Python
This project demonstrates how to use SQLite + Python to analyze a small sales dataset


As a fresher, the goal of this task was to learn:

=> How to store data in an SQLite database

=> How to write basic SQL queries

=> How to load SQL results into Python using pandas

=> How to do simple data analysis

=> How to visualize revenue using a bar chart

=> This is a small but complete end-to-end data analysis workflow.
